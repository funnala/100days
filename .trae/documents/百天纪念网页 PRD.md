# 百天纪念网页 PRD 文档

## 1. 产品概述

一个浪漫的百天纪念网页，通过精美的翻页效果展示情侣间的甜蜜回忆。网页采用清新的绿色主题，营造温馨浪漫的氛围，让每一页都成为珍贵的纪念。

**目标用户：** 情侣（尤其是为女朋友准备的惊喜）
**核心价值：** 将百天纪念打造成一个充满仪式感的数字纪念品

## 2. 核心功能

### 2.1 页面结构

整个纪念网页包含多个翻页页面，涵盖以下内容模块：

1. **封面页**
   - 百天纪念标题
   - 情侣照片占位
   - 甜蜜的开始文案

2. **回忆展示页（3-5页）**
   - 照片展示区域（占位符，便于后期添加）
   - 温馨文案区域（占位符，便于后期添加）
   - 每页独立的视觉主题元素

3. **数据页**
   - 在一起的天数统计
   - 重要的里程碑事件

4. **情话页**
   - 对女朋友的甜蜜告白
   - 未来期许

5. **结束页**
   - 温馨的结尾语
   - 制作日期

### 2.2 交互功能

- **翻页动画**：流畅的3D翻书效果
- **键盘控制**：支持左右方向键翻页
- **触摸滑动**：移动端支持手势滑动
- **页面指示器**：显示当前页码和总页数
- **自动播放（可选）**：可开启自动翻页模式

## 3. 设计规范

### 3.1 色彩方案（绿色主题）

**主色调：**
- 薄荷绿（#98D8AA）- 清新主色
- 森林绿（#228B22）- 深色强调
- 浅绿（#90EE90）- 背景辅助

**辅助色：**
- 奶油白（#FFFEF0）- 温暖背景
- 玫瑰粉（#FFB6C1）- 浪漫点缀
- 金色（#FFD700）- 重要元素高亮

### 3.2 字体选择

- **标题字体**：使用优雅的手写风格字体，如 "Ma Shan Zheng" 或 "ZCOOL XiaoWei"
- **正文字体**：使用清晰易读的中文字体，如 "Noto Sans SC"
- **英文装饰**：使用优雅的衬线字体

### 3.3 布局风格

- **整体风格**：现代简约 + 浪漫温馨
- **卡片设计**：圆角卡片，轻微阴影
- **间距**：大量留白，营造呼吸感
- **对齐**：居中对齐为主，符合阅读习惯

### 3.4 动画效果

- **翻页动画**：3D透视翻书效果，0.8s时长
- **页面进入**：淡入上移效果，带有轻微缩放
- **文字动画**：逐字显示，增强仪式感
- **装饰动画**：飘落的花瓣或树叶（绿色主题）

## 4. 技术实现

### 4.1 技术栈

- **框架**：纯 HTML5 + CSS3 + JavaScript（无复杂依赖）
- **动画库**：使用 CSS 3D transforms 实现翻页效果
- **字体**：Google Fonts 中文字体
- **图标**：使用 emoji 或 SVG 图标

### 4.2 响应式设计

- **桌面端**：大屏幕展示最佳效果
- **平板端**：自适应布局，保持翻页体验
- **手机端**：优化触摸交互，支持全屏滑动

### 4.3 文件结构

```

  ├── css/

  ├── js/

```

## 5. 内容占位说明

为便于后期添加内容，将以下区域设计为占位符样式：

1. **照片区域**：灰色占位框 + 上传提示文字
2. **文案区域**：示例情话文本，可直接替换
3. **日期数据**：预设占位日期，用户可修改

所有占位符都包含清晰的注释说明，方便快速定位和替换。
