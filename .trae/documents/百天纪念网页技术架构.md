## 2. 技术描述

### 2.1 技术栈

- **HTML5**：语义化结构
- **CSS3**：
  - CSS Variables（主题变量）
  - 3D Transforms（翻页效果）
  - Flexbox & Grid（响应式布局）
  - Animations & Transitions（动画效果）
- **Vanilla JavaScript (ES6+)**：
  - 事件处理
  - DOM 操作
  - 动画控制

### 2.2 第三方资源

**字体库：**
- Google Fonts - Ma Shan Zheng（手写标题）
- Google Fonts - Noto Sans SC（中文正文）

**无外部 JS 依赖**，全部使用原生实现

## 3. 页面路由

本项目为单页应用，无需路由配置。页面通过 JavaScript 控制显示/隐藏。

```
index.html
  ├── 页面切换逻辑
  ├── 动画控制系统
  └── 交互事件处理
```

## 5. 核心模块设计

### 5.1 HTML 结构

```html
<!-- 翻页容器 -->
<div class="book" id="book">
  <!-- 每一页 -->
  <div class="page" data-page="1">
    <!-- 页面内容 -->
  </div>
</div>

<!-- 导航控制 -->
<div class="controls">
  <button id="prevBtn">上一页</button>
  <span id="pageIndicator">1 / 10</span>
  <button id="nextBtn">下一页</button>
</div>
```

### 5.2 CSS 架构

**变量定义：**
```css
:root {
  --mint-green: #98D8AA;
  --forest-green: #228B22;
  --light-green: #90EE90;
  --cream-white: #FFFEF0;
  --rose-pink: #FFB6C1;
  --gold: #FFD700;
}
```

**翻页动画：**
```css
.page {
  transform-style: preserve-3d;
  transition: transform 0.8s cubic-bezier(0.645, 0.045, 0.355, 1);
}

.page.flip {
  transform: rotateY(-180deg);
}
```

### 5.3 JavaScript 模块

**核心功能：**
1. `initBook()` - 初始化翻页系统
2. `goToPage(pageNum)` - 跳转指定页
3. `nextPage()` - 下一页
4. `prevPage()` - 上一页
5. `handleKeyboard()` - 键盘事件
6. `handleTouch()` - 触摸滑动
7. `updatePageIndicator()` - 更新页码显示

## 6. 性能优化

- **CSS 动画**：使用 GPU 加速（transform, opacity）
- **懒加载**：照片使用懒加载
- **事件委托**：减少事件监听器数量
- **防抖处理**：避免快速点击导致动画异常
